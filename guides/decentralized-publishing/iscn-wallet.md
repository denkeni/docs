# ISCN Wallet

[ISCN Wallet](http://iscn.catding.tw/) 用於註冊、更新 ISCN、轉讓 ISCN、依照持有者地址列出 ISCN 及依照 ID 查找 ISCN。

使用 Vue.js + cosmjs + iscn-js 構建的 ISCN Wallet 是 [開源程序](https://github.com/catdingding/iscn-wallet)，歡迎參與改進。
